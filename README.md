# Basic Webservice built in Golang

## Run the web service

* If using windows, generate executable using the following command:
	```
		cd <ProjectLocation>

		go build .
	```

	run the `.exe`:
	```
		webservice.exe
	```

* Otherwise run the webservice using:
	```
		go run pluralsight/webservice
	```
Test the app in Postman.
